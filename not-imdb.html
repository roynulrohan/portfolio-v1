Not-IMDb

Not-IMDb is a movie database web application which allows users to browse through movies and review them. The app uses a database containing movie and user information. User authentication is supported and required to write reviews.

The app was developed with React for the front-end and Express for the REST API. The database used was MongoDB which allowed everything to be stored in JSON format and make several queries for CRUD functionalities.


Front-End: React, Redux, SASS, Bootstrap
Back-End: Node, Express, MongoDB
Other: Git, Heroku
